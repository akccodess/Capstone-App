# Vue App

## Install

```bash
npm install
```

## Environment

1. Copy `.env.example` to `.env`.
2. Update `.env` with your MySQL credentials.

## MySQL setup

Run the SQL in `server/schema.sql` using your MySQL client.

## Run development server

```bash
npm run dev
```

## Run Node backend server

```bash
npm run dev:server
```

## Run frontend + backend together

```bash
npm run dev:full
```

## Build for production

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```
